HTML

Markup ენა: HTML არის markup ენა, რაც ნიშნავს, რომ ის "ნიშნავს" ან "შენიშვნებს" დოკუმენტის შინაარსს, რომ ვებ ბრაუზერები (როგორც Chrome, Firefox, Safari) შეძლონ მისი სწორად გაშლა და ჩვენება. HTML არ არის პროგრამირების ენა, რადგან ის არ შეიცავს ლოგიკურ ოპერაციებს, როგორიცაა ციკლები, პირობები და ფუნქციები. სტრუქტურა: HTML-ს აქვს გარკვეული სტრუქტურა, სადაც შინაარსი ორგანიზებულია ელემენტების საშუალებით. თითოეული ელემენტი იღებს სახეს და ფუნქციას, რაც ეხმარება ბრაუზერს სწორად გაჩვენოს გვერდი. თეგები: HTML იყენებს თეგებს, რომლებიც ალაგებს შინაარსს. ტეგები ჩასმულია < და > ბრეკეტებში, მაგალითად,

CSS

CSS (Cascading Style Sheets) არის სტილიის ენა, რომელიც გამოიყენება HTML დოკუმენტების ვიზუალური და გრაფიკული პრეზენტაციის გასაუმჯობესებლად. CSS საშუალებას აძლევს დეველოპერებს განსაზღვრონ, თუ როგორ უნდა გამოიყურებოდეს HTML ელემენტები ბრაუზერში (მაგალითად, ფერები, შრიფტები, მარჟინები, საზღვრები, განლაგება და სხვა). CSS-ის ძირითადი ფუნქცია: CSS-ის მთავარი ფუნქციაა HTML დოკუმენტის ვიზუალის და სტილის განსაზღვრა, ამიტომ HTML პასუხისმგებელია შინაარსის სტრუქტურაზე (ტექსტი, სურათები, ბმულები), ხოლო CSS უზრუნველყოფს, რომ ეს შინაარსი სწორად და ლამაზად გამოჩნდეს.

javascript

JavaScript (JS) არის პროგრამირების ენა, რომელიც გამოიყენება ვებ გვერდების ფუნქციონალობის გასაუმჯობესებლად. ის ერთ-ერთი ყველაზე პოპულარული და ფართოდ გამოყენებადი ენაა, რომელიც გაძლევთ საშუალებას შექმნათ ინტერაქტიული და დინამიური ვებ გვერდები. JavaScript შეიძლება გამოიყენებოდეს როგორც ბრაუზერში (კლაიმენტში), ასევე სერვერის მხარეს (Node.js-ით).
